# Browserstack Tier Usage Dashboard

## Prerequisites

- [Node.js](https://nodejs.org/) installed

## Installation

Install the dependencies:

```bash
npm install

```

## Starting the Server

```bash
npm start
```

or

```bash
node server.js
```

## Running the App

- Open index.html on a browser
- Feel free to make changes to index.html based on your preferences.
